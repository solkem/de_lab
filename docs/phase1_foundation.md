# Phase 1 Foundation

This milestone establishes the minimum product skeleton for LakeForge:

- repo structure
- local infrastructure
- FastAPI control plane
- seeded lakehouse namespace model
- Next.js workspace shell
- Spark runtime foundation
- compute resource metadata
- table registration metadata APIs
- Delta-backed table materialization
- table detail and history inspection
- notebook metadata and execution APIs
- notebook detail, run history, and web workbench
- job metadata, runs, and web jobs surface
- pipeline metadata, target namespaces, and web pipelines surface
- pipeline expectations and lineage edges

Next milestone targets:

- system tables and operational observability
- richer notebook editor and multi-cell support
- richer table metadata CRUD
- notebook execution path
